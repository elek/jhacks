eXist

Nyílt forrású natív XML adatbázis. Natív alatt azt kell érteni, hogy az adattárolás történik XML-ben.

(http://exist.sourceforge.net/)

eXist - Open Source Native XML Database

Sok mindenre használható ... egyenlőre sajnos komoly hiányosságai vannak.

Lekérdezés és hozzáférés mehet sok formában:

  • XML:DB kezdeményezés API-ját használva,
  • REST stílusú HTTP-vel (thanks to Jetty),
  • XML-RPC-vel,
  • SOAP alapú webszolgáltatásokkal,
  • WebDAV-val ... ez a része nagyon komoly ... sajnos még nem teljes
  • XQuery-vel (majdnem teljes) ... esetleg megtámogatva:
  • XPath-tal,
  • dokument és csomópont szintű update XUpdate-tel.

Komoly hiányosságok:

  • nincs tranzakció kezelés,
  • nincs schema validálás ... meg egyáltalán schema kezelés,
  • az XQuery támogatás egyenlőre 97%-os ... ráadásul a keménymagos funkciók out of scope.

Ha lib-ként használod, alapból eléred mind az alap, mind az XQuery-vel előállított XML doksi DOM-ját. Néhány óra doksi olvasás után be tudtam tolni JonAS alá ... onnantól kezdve a web konténerből mindenféle úton-módon lehet használni. EJB konténerről egyenlőre ne beszéljünk ... amúgy sincs tranzakciókezelés ...
