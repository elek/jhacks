Harmony

(http://harmony.apache.org/)

Az ASF java VM projecje, ami java 1.5 és java 1.6 kompatibilis. Már letölthető milestone csomagok is vannak benne, szóval dolgoznak a srácok keményen, még ha csendben is :) Amikor elkezdték a projectet, a mai OpenJDK még nem létezett. Az OpenJDK létrejötte jórészt kifogta a harmony project vitorlájából a szelet, de nem teljesen azért...

Legutóbb egy eclipse felstartolásával próbáltam ki, hogy működik-e, és bár elég látványosan lassabb volt mint az OpenJDK, hiba nem volt a futása során.
