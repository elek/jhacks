Eclipse Pollinate Plugin

eclipse plugin az Apache beehive projecthez. Sajnos kicsit megakadt a fejlesztese ugy tunik, a eclipse 3.1m5-hoz es regi eclipse/plug-in/wtphez van csak release :(

Eszrevetelek:

  • csak java 1.5-tel megy, marminthogy az eclipsedet is ugy kell elinditani, nem csak a projectet
  • A page flow editor az egy az egyben ugyanaz lenne mint a bea weblogic workshop cucca, csak sajnos olyan alap dolgok nem mukodnek benne mint oldalak vagy actionok atnevezese. Bar duplakattintasra szepen feljon maga az oldal.

Na mind1, szoval jelenleg csipkerozsika almat alussza.
