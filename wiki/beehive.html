beehive

Apache Beehive

Ez a Project az Apache zaszloja alatt egy J2EE es a Struts otvozeset tuzte ki celul. A lapok kapcsolatait Flow diagrammon keresztul adhatjuk meg. Jelenleg inkubator allapotban leledzik (Alpha V1) de igen figyelemremelto.

Eclipse-hez keszult mar hozza egy pollinate nevu editor plugin, ami nem mellesleg az Eclipse Technology Project egisze alatt fut, tehat nem valami kis csoport altal fejlesztett kulso plugin.

Az alapgondolata valami olyasmi hogy Contol-okat hozol letre, ezeket az actionok parameterezik fel es adjak at a jsp-nek. Szoval a jsp egyszeruen felhasznalhatja ezeket, es nem kell bele java kodot irni.

A masik dolga a page flow, ami egy kicsit kodos hogy hogy muxik :)

De mindenesetre mindezt java 1.5 annotations supporttal teszi.
