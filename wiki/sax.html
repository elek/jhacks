Simple Api for XML

( pont ugy kell kimondani mint azt hogy 'sucks', nem minden ok nelkul :) )

Egy api ami benne van a jdk-ban mar jo ideje es akkor celszeru hasznalni ha nagyon nagy XML allomanyokat akarsz feldolgozni. Nem hoz letre egy reprezentacios modellt mint a DOM, hanem eventeket kuldozget a listenerednek. Igy megtakarit egy csomo memoriat, meg dolgozhatsz vele egy csomot.

Tipikus push parser, azaz ha elkezd parsolni, akkor nyomja bele az eventeket a listenerednek, amig csak tart a fajl.

Hasonloan hatekony, csak pull parser megoldas a StAX
