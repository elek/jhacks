JSP

JavaServer Pages

A JSP a PHP es az ASP konkurenciajakent jott letre, illetve hogy a szervlet keszites automatizalhato es vizualis eszkozok szamara is lekepezheto legyen. A JSP lapokbol a web kontener szervleteket general majd ezeket futtatja a web kontener, igy elso futaskor a sebesseg lassu de a tobbi futasnal a sebesseg jelentossen no.

A legtobb vizualis java editor kepes JSP lapok szerkesztesere. A Vezerleshez (amennyiben nem MVC modellben hasznaljuk), kiterjedt tag library-k leteznek (adatbaziskezeles (JDBC), JNDI, Regexp, Session stb..), illetve sajat rutinkonyvtarak is keszithetok. (Azt azert hozzatennem hogy talan jobb ha a tagek is csak a megjelenitest szolgaljak, actionkent hasznalva piszkosul nagy szivasokra van kilatas (kocka))

Nehany tag library lelohely:

  • Java Standard TagLibrary
  • Jakarta taglibs
  • http://jsptags.com/tags/index.jsp
  • http://www.jspin.com/home/tags
  • http://www.servletsuite.com/jsp.htm
