A PL-J beta statuszban van. Csak a legbatrabb harcosoknak!

A PL-J egy fejlodoben levo java proceduralis nyelv implementacio archtekturalisan barmilyen RDBMS-re, jelenleg csak PostgreSQL 7.4.x-re.

Jelenleg kepes UDF-ek es triggerek kezelesere, a JDBC meghajto fejlesztese folyamatban.

Szinten avalon komponensekbol osszeallitott alkalmazas.

Loom containerben fut, ha nagyon akarod valahogy mukodesre birhatod avalon/merlin alatt is.

  • web: http://plj.codehaus.org
  • irc: irc://irc.codehaus.org/plj
  • ashkelon: http://ashkelon.forgeahead.hu/apis.do
  • build howto: http://docs.codehaus.org/display/PLJ/Build+and+install+howto
  • change log: http://docs.codehaus.org/display/PLJ/ChangeLog

A PL-J project tavlatokban (A JDBC fejezet utan):

  • ETC BSF integracio (keszuloben)
  • Groovy integracio
  • sajat, ujgeneracios fel-procedurails nyelv
  • Aszinkron/szinkron rendszerintegracio 2-tier rendszerkhez (JTA, JMS)
  • harvest integracio
  • eclipse plugin a fejlesztesek konnyitesere.
