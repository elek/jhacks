drone

http://drone.codehaus.org/

Irc bot tisztan java nyelven, deployolhato webappkent is, megy a sajat kis beepitett jetty szerverevel vagy standaloneban. A java webappok gyerekbetegsegeival megaldva :)

Belulrol egy kicsit erdekes dolgai vannak monolitikus a konfiguracioja, de a modul apija tetszik. A javadoccal se bibelodott sokat aki irta :)

A codehaus es a freenode szobait rogziti, itt talalhato: http://servlet.uwyn.com/drone/
