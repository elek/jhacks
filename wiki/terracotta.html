terracotta

(http://www.terracotta.org/)

Java virtualis gep clusterezesere szolgalo kiegeszites a jdkhoz. Szoval nem api mint a jcache, a terracotta egyszeruen az objektumok allapotat clusterezi. Sajat gc is van beleepitve, van hozza eclipse eclipse/plug-in, doksi is eleg jonak tunik. Tobbnyire a java 1.5 virtualis gepeket tamogatja, a java 1.4-et helyenkent.
