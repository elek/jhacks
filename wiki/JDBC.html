Elsore az ember azt hinne hogy a Java DataBase Connectivity roviditese lehet, de nem, a JDBC[TM](Missing.html) a sun bejegyzett vedjegye. :)
A legalapabb megoldas RDBMSek eleresere. Sajnos mivel az SQL szabvanyt mindenki maga modjan implementalta, a JDBC api kozvetlen hasznalata tobbnyire faszan RDBMS vendor fuggove teszi az alkalmazast. Ha mar JDBC-t hasznalasz, tegyel ala egy DAO reteget!

Alternativ megoldasok: OR Mapping, EJB

  • Oracle SQL Developer (JDBC, de csak oracle-hez)
