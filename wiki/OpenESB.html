OpenESB

https://open-esb.dev.java.net/

A Sun nyílt forráskódú JBI megoldása. Az 1-es változat csak glassfish-ben létezik, viszont a Netbeans nagyon jól támogatja (Enterprise Pack kell hozzá). A 2-es változatnak már van elvileg standalone futtatható (expreimental) változata is.

A core projekt mellett fut jónéhány alprojekt, amik kiegészítő BC és SC komponenseket fejlesztenek (SNTMP, SMTP, UDDI, FTP, XMMP, stb.)

A következő 3-as verzó, már nagyon moduláris lesz OSGi, a core-ját Project Fuji néven kezdik fejleszteni, és várhatóan mindenen futni fog:

"Open ESB v3 will be supporting various application server containers, like GlassFish, JBoss, WebSphere, WebLogic, etc, but also runs standalone on Java SE. In the Java SE case, all you need is an OSGi container like Apache Felix"
