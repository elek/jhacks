OpenOffice.org

(http://OpenOffice.org/)

Open source szovegszerkeszto, spreadsheet, bemutatokezelo, meg miegymas foleg suntol csak hogy a microsoft arcok elete se legyen csak jatek es mese :)

Mitol olyan ql:

  • RDBMS integracio (PostgreSQL es MySQL backend tamogatott, elvileg barmi mas JDBC, vagy egyeb adatforrasok)
  • Java integracio
  • Az egesz XML alapu, ugyhogy nem fogunk sirni az idiota fileformatum miatt. Vagy pl hogy meg sajat magat se ertene idonkent mint az MS-WORD :)
