GCJ

(http://gcc.gnu.org/java/)

GNU compiler for java

Nagyon trendi, de szvsz alapvetöen elvetélt elképzelés, ugyanis egy elég dinamikus nyelvet nagyon nehéz statikusra fordítani. De ennek ellenére próbálják sokan.

Ehhez kapcsolódó projektek:

  • mod_gcj Apache module szervlet-szerüségek futtatására
  • Classpath a GNU osztálykönyvtár,deklarációs szinten elég teljes (95%<), viszont jónéhány, vagy hiányos implementációval. Azért eclipse-t már lehet rajta futtatni.
