python

http://www.python.org/

(Monthy Python utan)
Egy nagyon klassz kis scriptes nyelv, viszonylag gyors, objektum orientalt de hagyomanyos spagetti kodot is konnyen lehet vele gyartani, igy egyarant alkalmas kis scriptes feladatokra, es komplexebb rendszerek fejlesztesere. A javahoz hasonloan platfrom fuggetlen, mivel nincs virtualis gepe, sokkal gyorsabban startol el mint egy java program, hardware szuksegletei is minimalisak. Eleg jo az exception kezelese (kb mint a java).

Bar izles dolga, de uzleti logika megfoglamazasara eleg jol hasznalhato, foleg annal fogva hogy akarhova be lehet agyazni. Pl: pl/python a postgresql-ben tarolt eljaras nyelvkent, jre-n futo verzioi reven alkalmazas szerverekben, vagy barhol ahol java futhat, es shelles implementacioja reven unixes windows kornyezetekben.

Maganvelemeny, de nagy rendszerek epitesenel csak rugalmas alkalmazaslogikat irnam pythonban, hianyolom belole azt a rengeteg api-t ami a javaban mar megvan.
