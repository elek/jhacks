Derby

Apache Derby project

Az IBM megosztotta a CloudScape forraskodjat, szabadda teszi (ASF) es ebbol lett a Derby. (Tehat egy kereskedelmi cucc Open Source forkja) XA kompatibilis, mukodik hibernatetel, peldaul testhez kivallo megoldas, vagy kicsi webappokba agyazva is akar.

Az adminisztracios igenyei szinte zeroval egyenloek, adatbazist peldaul ugy hoz letre hogy a JDBC URL string vegere passzolsz egy parametert.

A Java 1.6 SDK-val szallitott Java DB alaja is ez a db.
