PL-J/DevTools

FORKOLVA

Ez a package kikerult a PL-J alol es alapos refactoring utan J4SQL lett az uj othona.

A PL-J DevTools egy kulon projectben keszulo kis taszkok (anthoz es mavenhez) keszlete lesz java tarolt eljarasok keszitesehez. Eddig igazabol a tervezes zajlik. Egy nagyon buta peldaval hogy mit fog csinalni:

CVS: A DevTools elerheto a PL-J CVSben.
web cvs: http://cvs.plj.codehaus.org/pl-j-devtools/

Jelenleg meg eleg minimalis funckionalitasa van, csak PostgreSQL-PL-J kombinacora.

public class JSProc1 {

/**
 * A very stupid UDF.
 * 
 * @jsproc.udf name="int_add"
 * @jsproc.return integer
 * @jsproc.param arg1 in integer
 * @jsproc.param arg2 in integer
 * @jsproc.properties immutable
 */
public static int add(int arg1, int arg2) {
 return arg1 + arg2;
}

}

Es ebbol neked legeneralja majd a kovetkezo SQL kodot PostgreSQL-hez es Pl-J-hez:


create function int_add(integer, integer) returns integer as
'
class=JSProc1
method=add
' language 'PL-J' immutable

comment on function int_add(integer, integer) is 'A very stupid UDF.';

Hat KB ennyi. Na most az extra az annyi, hogy nem csak PostgreSQL/PL-J-re generalna kodot, hanem mas DB-kre is, Db2, Oracle, satobbi...
