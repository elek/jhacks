Platform, nyelv, runtime

hupwiki - A legnagyobb kérdés az, hogy a Java most vajon mi? Igazabol nem tudom ki az aki ezt nem erti, de akkor tessek:

  1. A java egy oop nyelv.
  2. A java forraskodot bytecodera lehet leforditani es java futasi kornyezeten (JRE) futtatni.
  3. A JRE platformra nem a java az egyetlen nyelv, lásd groovy, scala, jruby...

Ezt gondolom az is tudta aki irta csak olyan jo egy kicsit ertetlenkedni :-D
