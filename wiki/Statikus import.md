Elenyésző újítása a [Java 1.5](java%201.5.html)-nek. Egy osztály statikus tagjait lehet vele importálni. 

Iskolapélda:
{% highlight java %}
import static java.lang.Math.*;
...
...
System.out.println(random());
{% endhighlight %}

Nem csomag, hanem osztály elemeire mondjuk.
